SimpleSketchJS

User Guide

About

SimpleSketchJS is a javascript based digital painting application. As the name suggests, the application features a concise toolset for users to create their paintings. However, the plan is to expand the application as the opportunity arises. Thank you for using SimpleSketchJS and enjoy!

This manual contains information regarding the features of SimpleSketchJS.

Tools

On the far left of the application is the Tool Pane.
Right beside is the application interface which includes a tool context menu.
Below are descriptions for each tool and their behavior.

Brush

Currently not implemented
A stroke based tool. The brush tool differs from the Pen and Pencil Tool by having a baked in multiply feature.

Eraser

A stroke based tool. Removes pixel data from a layer.

Eye Dropper

Current not implemented
Grabs a pixel color from a specific mouse point.

Pen

A stroke based tool. Performs strokes without any varience in opacity.

Brush

A stroke based tool. Looser then the Pen tool, reacts to pen pressure applied.

Magnify

Currently not implemented
Zoom's into a canvas based on the tool position. Holding Ctrl zoom's out.

Palette

On the left side of the application is a series of boxes intended for use as the user's palette.
Single clicking the box select that color for use. Doubling clicking will allow the user to customize the color.
A new panel will appear allowing the user to set the RGB values of the new color.
Once complete, the new color will be selected.

Hot Keys

b

Selects the Brush tool.

e

Selects the Eraser tool.

i

Selects the Eye Dropper tool.

p

Selects the Pen tool.

P

Selects the Pencil tool.

z

Selects the Zoom tool.

[

Decreases the size of the active stroke based tool.

]

Increases the size of the active stroke based tool.

{

Decreases the opacity of the current stroke based tool by 10.

}

Increases the opacity of the current stroke based tool by 10.
